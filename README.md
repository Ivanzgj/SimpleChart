# SimpleChart
一个自定义的图表，暂时只有折线图和堆叠柱状图以及平行柱状图三种，不过对于一般需求应该也够了。支持自定义主题（肤色），内置两种，分别是THEME_LIGHT和THEME_DARK。支持基于基类Chart的坐标系定制其它图表，Chart基类已提供了在其坐标系内绘制点，线，矩形的api。
